# NGDC Science Club Website

A fully responsive website for NGDC Science Club with admin panel for dynamic content management.

## Features

1. **Responsive Design** - Works perfectly on mobile, tablet, and desktop
2. **Admin Panel** - Secure login to manage all content
3. **Dynamic Content** - Update notices, executives, and text without coding
4. **Join Now System** - Direct Google Form integration
5. **Beautiful UI** - Modern design with animations

## Setup Instructions

### 1. Prepare Images

Create an `images` folder and add:

- `logo.png` - Club logo (recommended: 200x200px)
- `club-photo.jpg` - Hero section photo (recommended: 800x600px)
- `favicon.png` - Website icon

### 2. Update Google Form Link

In the admin panel, go to "Home Content" tab and update the "Join Now Link" with your actual Google Form link.

### 3. Update Social Links

In the admin panel, go to "Footer" tab and update:

- Facebook Page Link
- Instagram Link
- Club Email

### 4. Add Initial Content

Using the admin panel (username: `ngdcsc`, password: `ngdc@sc@2026#`), add:

- Club executives with photos, names, positions, contact info
- Current notices and events
- Club description and mission statement

### 5. Deploy to Vercel

1. Create a GitHub repository
2. Push all files to GitHub
3. Go to Vercel.com
4. Import your GitHub repository
5. Deploy with default settings
